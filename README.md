# paging

This is an example of using paging where the positions of items on the screen are saved and restored after returning.

Used:
  - Splash screen
  - Paging library
  - Navigation component
  - Retrofit
  - Room
  - Dagger
  - Coroutines
  - Saved State module for ViewModel
  - Timber
  - Stetho
  - Fake server response
